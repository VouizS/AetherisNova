# Aetheris Nova 0.1.2 — Sprint 02 Browser Interaction Core

## Objetivo

Melhorar o comportamento real do navegador sem abandonar a base visual Glass Orbit aprovada na 0.1.1.

## Implementado

- Barra de progresso de carregamento.
- Estado/título da página abaixo da topbar.
- Melhor comportamento do botão voltar.
- Melhor comportamento do botão home.
- Melhor comportamento do botão recarregar.
- Menu inferior expandido:
  - Nova busca
  - Copiar link
  - Compartilhar
  - Recarregar
  - Abrir downloads
  - Estado da página
  - Sobre
- Busca com atalhos:
  - google
  - youtube
  - github
  - duck
- Tratamento melhorado de erro de página.
- Downloads continuam ativos.
- Permissões WebRTC base continuam ativas.
