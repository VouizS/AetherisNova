# Aetheris Nova 0.1.0 — Sprint 00 Foundation

## Decisão

A base `Aetheris-FOSS-Base` foi arquivada como referência. O Aetheris passa a seguir uma base nova, criada do zero para uma interface própria.

## Objetivo

Criar a fundação limpa do Aetheris Nova.

## Implementado

- Projeto Android novo.
- Java Activity sem AppCompat para reduzir complexidade.
- WebView funcional.
- Interface própria em runtime.
- Home própria do Aetheris Nova.
- Campo de busca/endereço.
- Bottom bar própria.
- Preparação de permissões de câmera/microfone.
- Tratamento de links externos básicos.
- Reuso do ícone oficial aprovado, se disponível na base antiga.
- GitHub Actions.
- Artifact do APK.
- Artifact de backup da source.
