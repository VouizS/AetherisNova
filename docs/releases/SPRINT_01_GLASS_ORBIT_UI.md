# Aetheris Nova 0.1.1 — Sprint 01 Glass Orbit UI

## Objetivo

Substituir a primeira interface simples da fundação por uma interface própria inspirada em design systems reais, com estética glass/floating, sem copiar iOS/Safari.

## Direção

Aetheris Glass Orbit UI:

- Barra superior compacta e flutuante.
- Home redesenhada.
- Campo de busca em glass card.
- Tiles rápidos com legenda.
- Dock inferior flutuante.
- Menu em sheet customizado, sem AlertDialog cinza padrão.
- Visual premium, escuro e mais limpo.
- Sem blur real pesado para preservar performance em Android fraco.

## Implementado

- MainActivity reescrita com nova interface programática.
- Topbar mais compacta.
- Home com Glass Orbit visual.
- Dock inferior em capsule.
- Menu custom sheet.
- Compartilhar link.
- Mantidas funções:
  - WebView
  - busca/endereço
  - voltar/avançar
  - home
  - recarregar
  - download básico
  - permissões WebRTC base
  - links externos
