# Aetheris Nova

Aetheris Nova é a nova base oficial do navegador Aetheris.

Esta versão abandona a base FOSS antiga como base principal e inicia uma fundação limpa, feita para identidade visual própria desde o começo.

## Versão

`0.1.0-foundation`

## Sprint

Sprint 00 — New Foundation

## Recursos iniciais

- Android Java limpo.
- WebView funcional.
- Interface própria programática.
- Tela inicial Aetheris Nova.
- Barra superior com campo de busca/endereço.
- Barra inferior com voltar, avançar, home, recarregar e menu.
- Preparação para câmera/microfone WebRTC.
- Tratamento inicial de links externos.
- DownloadManager básico.
- GitHub Actions para build do APK.
- Artifact de APK e backup de source.

## Base antiga

A pasta `Aetheris-FOSS-Base` fica como referência técnica/backup, não como base principal futura.
